# **Validador de CPF**

 Biblioteca em javascript para verificar se um número de cpf é válido. 

---

 ### Como instalar: 
 
>  $  npm install cpfvalidatorln 

### Como utilizar:

> const validate = require("cpfValidatorln");

> validate.cpfValidator('31306573041')

> // returns "true"
